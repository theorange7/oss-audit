## Agent skills

### Issue tracker

Issues are tracked in the `theorange7/oss-audit` GitHub Issues via the `gh` CLI. External PRs are *not* a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical default labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
